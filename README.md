# Finance_Analytics_Dawa

This project demonstrates **financial analytics** for a pharmaceutical company in Kenya.  

**Objectives:**  
- Track **drug sales revenue and costs**  
- Analyze **profit per drug and per batch**  
- Monitor **departmental expenses**  
- Provide insights for **budgeting and financial decision-making**  

**Skills demonstrated:** SQL, Finance Analytics, Revenue & Cost Tracking, Profitability Analysis  

**Drugs included:**  
- Cefadroxil 500mg  
- Diclofenac 50mg  
- Metformin 500mg  

**Queries covered:**  
1. Total revenue per drug  
2. Profit per drug  
3. Total departmental expenses  
4. Monthly revenue summary  
5. Profit margin per drug
